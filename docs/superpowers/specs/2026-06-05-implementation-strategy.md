# AI 同声传译助手 — 实施策略

> 版本: v1.0 | 日期: 2026-06-05 | 基于设计文档 v1.1

---

## 实施方式: 垂直切片

按功能链路逐步打通，每个切片产出可演示的增量功能。

### Slice 1: 项目脚手架 + 连通性验证

```
音频捕获(浏览器API) → WebSocket(收发验证) → 字幕占位(纯文本)
```

| 层 | 内容 |
|----|------|
| 后端 | FastAPI 项目骨架、WebSocket 端点、config 模块、消息协议定义 |
| 前端 | Vite + React + TS 项目骨架、WebSocket 客户端、TabCapture 音频捕获、简易字幕浮层 |

产出: 两端项目跑通，音频帧成功传输，收到回显

### Slice 2: ASR 集成

```
音频捕获 → Deepgram 流式 ASR → 英文字幕
```

产出: 英文语音 → 英文实时字幕

### Slice 3: 翻译集成

```
音频捕获 + 过滤预处理 → ASR + DeepSeek 流式翻译 → 双语字幕
```

产出: 英文语音 → 中英双语实时字幕 + Interim 过滤

### Slice 4: TTS + 修正引擎

产出: 中文语音朗读 + 翻译回溯修正 + 修正动画

### Slice 5: 完善

产出: 设置面板、降级切换、连接预热、影院模式
